# Changelog

## 1.0.7

- Changed the default Day price to €0.25/kWh.
- Changed Night, Nightboost, and all previous prices to €0.00/kWh.
- Changed the default discount and VAT to 0%.

## 1.0.6

- Added configurable Day, Night, and Nightboost tariff times.
- Added Night tariff start and end controls.
- Added optional yearly kWh and cost totals.
- Added options to hide the monthly calendar and tariff information.
- Automatically hides the kWh/currency selector when the calendar is hidden.

## 1.0.5

- Added yearly totals below the monthly calendar.

## 1.0.4

- Added improved tariff and price configuration.

## 1.0.3

- Added HACS metadata and package documentation.

## 1.0.0

- Initial release.
